名前

xzgrep - 正規表現を使って圧縮ファイルを検索  

書式

xzgrep [grep_options] [-e] pattern file...
xzegrep ...
xzfgrep ...
lzgrep ...
lzegrep ...
lzfgrep ...  

説明

xzgrep は以下の圧縮形式 xz(1), lzma(1), gzip(1), bzip2(1), lzop(1) によって圧縮されている、あるいは圧縮されていない files に対して grep(1) を呼び出します。指定されたオプションは、すべて直接 grep(1) に受け渡されます。

file の指定がなかった場合、標準入力からの情報が必要に応じて伸長されて、grep(1) に受け渡されます。標準入力から読み込む場合、gzip(1), bzip2(1), lzop(1) の圧縮ファイル形式はサポートされません。

xzgrep は xzegrep または xzfgrep として呼び出される場合は、grep(1) の代わりに egrep(1) や fgrep(1) が用いられます。lzgrep, lzegrep, lzfgrep として呼び出される場合も同様であり、これらは LZMA Utils との後方互換性のために提供されています。

 

環境変数

GREP
環境変数 GREP が設定されている場合は、xzgrep は grep(1), egrep(1), fgrep(1) ではなく、その変数値を利用します。
 

関連項目
