名前

ethers - イーサーネットアドレスを IP 番号に変換するデータベース  

説明

/etc/ethers は、 48 ビットのイーサーネットアドレスと、 それに対応する IP 番号とからなる。 IP 番号ごとに一行を書く。

Ethernet-address IP-number

二つの要素は任意の個数のスペース文字またはタブ文字によって区切る。 # で始まる行は、行末までコメントとみなされる。 Ethernet-address の書式は のようなものである。各 x は 0 から ff までの 16 進の整数で、アドレスの各 1 バイトをネットワークバイトオーダー (big エンディアン) で表す。 IP-number は DNS で解決可能なホスト名か、 ドット区切りの数値である。  

例

08:00:20:00:61:CA pal  

ファイル

/etc/ethers
