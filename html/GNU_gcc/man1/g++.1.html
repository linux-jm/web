名称

g++ - GNU プロジェクト C++ コンパイラ (v2.4)  

書式

g++ [option | filename ]...  

解説

C コンパイラおよび C++ コンパイラは統合されました。 g++ は gcc に C++ を解釈するようにするオプションをつけてコールするスクリプトです。 詳細は英語版のオンラインマニュアルおよび gcc(1) を参照して下さい。
