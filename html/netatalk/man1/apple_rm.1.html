APPLE_RM

Section: User Commands (1)
Updated: 16 Oct 2001
Index JM Home Page roff page
 

名前

apple_rm - apple ファイルを削除し、リソースフォークも削除する

 

書式

/usr/bin/apple_rm FILE...

 

説明

apple_rm は perl スクリプトであり、 FILE とそれに対応する .AppleDouble リソースフォークファイルを削除する。 apple_rm はディレクトリを削除しない。

 

例

/usr/bin/apple_rm test.text

/usr/bin/apple_rm test.text testing.text

 

バグ報告

バグは Netatalk-devel メーリングリスト <netatalk-devel@lists.sourceforge.net> へ報告してください。

 

関連項目

apple_cp(1), apple_mv(1).
