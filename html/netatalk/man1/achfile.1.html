名前

achfile - Apple Macintosh ファイル (netatalk フォーマット) のタイプとクリエータを変更する  

書式

achfile [ -t type ] [ -c creator ] file ...

 

説明

achfile は、引き数 file の Macintosh タイプとクリエータを変更する。 file には対応する .AppleDouble ファイルが存在しなければならない。

 

オプション

-t type
タイプを変更する。

-c creator
クリエータを変更する。

 

返り値

全てのファイルの変更が成功した場合は、終了状態 0 を返す。  

関連項目

afile(1), afpd(1)
