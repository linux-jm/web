operator - C 言語の演算子の優先順位と評価の順序  

説明

この man ページでは C 言語の演算子と評価の優先順位をリストする。 

演算子                                結合の順序
() [] -> .                           左から右へ
! ~ ++ -- + - (type) * & sizeof      右から左へ
* / %                                左から右へ
+ -                                  左から右へ
<< >>                                左から右へ
< <= > >=                            左から右へ
== !=                                左から右へ
&                                    左から右へ
^                                    左から右へ
|                                    左から右へ
&&                                   左から右へ
||                                   左から右へ
?:                                   右から左へ
= += -= *= /= %= <<= >>= &= ^= |=    右から左へ
,                                    左から右へ
