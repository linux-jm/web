名前

iswctype - ワイド文字の分類  

書式

#include <wctype.h>

int iswctype(wint_t wc, wctype_t desc);
 

説明

wc が desc で指示された文字属性を持つ (言い換えると、desc で指示 された文字クラスに属する) ワイド文字ならば、 iswctype() 関数は 0 で ない値を返す。それ以外の場合にはこの関数は 0 を返す。wc が WEOF ならば 0 が返される。

desc は wctype(3) 関数が返す文字属性指示子でなければならない。  

返り値

iswctype() 関数は、wc が指示された属性を持つならば 0 でない 値を返す。そうでなければ 0 を返す。  

属性

この節で使用されている用語の説明については、 attributes(7) を参照。
インターフェース属性値
iswctype() Thread safetyMT-Safe
 

準拠

POSIX.1-2001, POSIX.1-2008, C99.  

注意

iswctype() の動作は、現在のロケールの LC_CTYPE カテゴリーに依存する。
