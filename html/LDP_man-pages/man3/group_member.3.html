名前

group_member - プロセスがグループに属しているかを検査する  

書式

#include <unistd.h>

int group_member(gid_t gid);

glibc 向けの機能検査マクロの要件 (feature_test_macros(7) 参照):

group_member(): _GNU_SOURCE  

説明

group_member() 関数は、 呼び出し元の補助グループ ID (getgroups(2) が返す値) のいずれかが gid に一致するかを検索する。  

返り値

group_member() 関数は、 呼び出し元の補助グループ ID のいずれかが gid に一致すれば 0 以外の値を、そうでなければ 0 を返す。  

準拠

この関数は非標準の GNU 拡張である。  

関連項目

getgid(2), getgroups(2), getgrouplist(3), group(5)
