名前

fputws - ワイド文字の文字列を FILE ストリームに書き込む  

書式

#include <wchar.h>

int fputws(const wchar_t *ws, FILE *stream);
 

説明

fputws() 関数は、 fputs(3) に対応するワイド文字関数である。 この関数は、ws から始まるワイド文字列を終端の L'\0' まで stream に書き出す。ただし終端のヌルワイド文字 (L'\0') は含まれない。

これらの処理を停止せずに行いたいときは、 unlocked_stdio(3) を参照のこと。  

返り値

fputws() は、操作が成功した時には負でない整数値を返す。エラーの時 には -1 を返す。  

属性

この節で使用されている用語の説明については、 attributes(7) を参照。
インターフェース属性値
fputws() Thread safetyMT-Safe

 

準拠

POSIX.1-2001, POSIX.1-2008, C99.  

注意

fputws() の動作は、現在のロケールの LC_CTYPE カテゴリーに依存する。

fopen(3) システムコールに渡す追加情報がない場合には、 fputws() が実際 にはワイド文字の文字列である ws に対応するマルチバイトシーケンス を書き込むと期待することは適切である。  

関連項目

fputwc(3), unlocked_stdio(3)
