名前

strtoimax, strtoumax - 文字列を整数に変換する  

書式

#include <inttypes.h>

intmax_t strtoimax(const char *nptr, char **endptr, int base);
uintmax_t strtoumax(const char *nptr, char **endptr, int base);
 

説明

これらの関数はちょうど strtol(3) と strtoul(3) に似ているが、それぞれ intmax_t と uintmax_t 型の値を返す。  

返り値

成功した場合、変換された値が返される。 変換するものが見つからなかった場合、0 が返される。 オーバーフローまたはアンダーフローの場合、 INTMAX_MAX または INTMAX_MIN または UINTMAX_MAX が返され、 errno が ERANGE に設定される。  

属性

この節で使用されている用語の説明については、 attributes(7) を参照。
インターフェース属性値
strtoimax(), strtoumax() Thread safetyMT-Safe locale
 

準拠

POSIX.1-2001, POSIX.1-2008, C99.
