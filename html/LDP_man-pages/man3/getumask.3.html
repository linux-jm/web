名前

getumask - ファイル作成マスクを取得する  

書式

#define _GNU_SOURCE /* feature_test_macros(7) 参照 */
#include <sys/types.h>
#include <sys/stat.h>

mode_t getumask(void);  

説明

この関数は現在のファイル作成マスクを返す。 基本的には以下と等価である。

mode_t getumask(void) {
    mode_t mask = umask( 0 );
    umask(mask);
    return mask; }

ドキュメントによると、この関数は スレッドセーフである (umask(2) ライブラリコールとロックを共有する) 点が異なる。  

準拠

ドキュメントに書いてあるだけの GNU 拡張である。  

注意

This function is documented in the glibc manual, but, as at glibc version 2.24, it is not implemented on Linux. (See umask(2) for a thread-safe method of discovering a process's umask.)
