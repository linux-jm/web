名前

ide_info - IDE デバイス記述ツール  

書式

ide_info device  

説明

ide_info は指定した IDE デバイスファイルをオープンし、 デバイスの識別情報を取得する。 このコマンドの出力は、シェル変数 MODEL, FM_REV, SERIAL_NO を取り出した情報に基づいて定義する、 Borne シェルのコマンド文字列になっている。 IDE 系のデバイス用の PCMCIA 設定スクリプトの中でこのユーティリティを使えば、 シリアル番号を使って特定のカードに対する設定を指定できる。
