名前

false - 無動作，異常終了  

書式

false
false {--help,--version}  

説明

このマニュアルは GNU 版の false （おそらく他のどの版でも同様）について説明する。 false は何もせず、終了ステータスとして``失敗''を意味する 1 を返す。 本コマンドはシェルスクリプト中で異常終了するコマンドが必要な場合に 使われる。  

オプション

GNU 版の false を単一引数で起動する場合、以下のオプションが認識される。
--help
利用法を標準出力に出力して終了する。
--version
本コマンドのバージョン情報を標準出力に出力して終了する。
