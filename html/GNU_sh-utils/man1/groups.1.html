名前

groups - ユーザーが属しているグループの表示  

書式

groups [username...]
groups {--help,--version}  

説明

このマニュアルページは GNU の groups のために書かれた。 groups は username で指定されたユーザが属するグループ名を表示する。 引数を省略すると、コマンドを入力したユーザの属するグループ名を 表示する。ユーザ名が指定されたときは、指定されたユーザの属するグルー プ名を表示する前にそのユーザ名が表示される。

グループのリストは `id -Gn' の実行結果に等しくなる。  

オプション

GNU の groups は一つだけ引数が与えられて実行されたとき，以下 のオプションが有効になる。
--help
標準出力に使用法を表示し，正常に終了する。
--version
バージョンを表示し，正常に終了する。
