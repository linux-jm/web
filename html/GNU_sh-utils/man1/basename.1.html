名前

basename - ファイル名からディレクトリとサフィックスを取り除く。  

書式

basename name [suffix]
basename {--help,--version}  

説明

このマニュアルページはGNUバージョンの basename について書いている。 basename コマンドはファイル名 name からディレクトリ名を取り去る。 suffix が指定されたり、 name の最後がこれと同一のものだと、 name からこれを取り去る。 basename はその処理をした name を標準出力に表示する。  

オプション

--help
使用法を標準出力に表示し、終了する。
--version
バージョン情報を標準出力に表示し、終了する。
