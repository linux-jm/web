名前

tload - システムの負荷をグラフィカルに表示する  

書式

tload [-V] [-s scale] [ -d delay ] [tty]  

説明

tload は現在のシステムの負荷 (load average) を、 指定された tty にグラフで表示する (指定されなければ tload プロセスの tty に表示する)。  

オプション

-s scale オプションは画面上の縦軸を設定する。 load average 1 に対応するグラフのひと目盛が、 scale 文字分になる。したがって小さい値を設定すると スケールは大きくなる。逆も同じ。

-d delay はグラフの更新間隔を秒単位で指定する。

 

ファイル

/proc/loadavg 負荷情報

 

関連項目

ps(1), top(1), uptime(1), w(1)

 

バグ

-d delay オプションは引き数の時間を alarm(2) に設定する。 -d 0 を指定すると alarm は 0 となり、 SIGALRM は決して送信されず、画面は更新されない。
