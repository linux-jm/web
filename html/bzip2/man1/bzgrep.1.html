BZGREP

Section: User Commands (1)
Index JM Home Page roff page
 

名前

bzgrep, bzfgrep, bzegrep - bzip2 圧縮されている可能性のあるファイルで、正規表現の検索をする  

書式

bzgrep [ grep_options ] [ -e ] pattern filename...
bzegrep [ egrep_options ] [ -e ] pattern filename...
bzfgrep [ fgrep_options ] [ -e ] pattern filename...  

説明

bzgrep は、 bzip2 で圧縮されたファイルに対して grep を呼び出すために使われます。 指定されたすべてのオプションは grep に直接渡されます。 ファイルが指定されなかった場合は、標準入力を (必要であれば伸長した後で) grep に渡します。 それ以外の場合は、指定されたファイルを (必要であれば伸長した後で) grep に渡します。

bzgrep が bzegrep や bzfgrep として呼び出された場合、 それぞれ egrep と fgrep が grep の代わりに使われます。 環境変数 GREP が設定されている場合、 bzgrep は grep プログラムとしてこの環境変数で指定されたものを呼び出します。 たとえば、


    sh の場合:  GREP=fgrep  bzgrep string files
    csh の場合: (setenv GREP fgrep; bzgrep string files)
