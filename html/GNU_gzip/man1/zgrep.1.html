ZGREP

Section: User Commands (1)
Index JM Home Page roff page
 

名前

zgrep - 圧縮されている可能性のあるファイルで、 正規表現の検索を行う  

書式

zgrep [ grep_options ] [ -e ] pattern filename...  

説明

zgrep コマンドは、 compress や gzip で圧縮されたファイルに対して grep を呼び出します。 指定されたオプションは、 すべて grep に直接渡されます。 ファイルが指定されない場合は、 標準入力を (必要であれば伸長した後で) grep に渡します。 それ以外の場合は、 指示されたファイルを (必要であれば伸長した後で) grep に渡します。

環境変数 GREP が設定されている場合、 zgrep は環境変数で指定された内容を grep プログラムとして呼び出します。  

終了ステータス

終了ステータスは、 マッチした場合 0、 マッチしなかった場合 1、 問題発生時は 2 を返します。  

バグ

以下の grep オプションはサポートされません： --dereference-recursive (-R), --directories (-d), --exclude, --exclude-from, --exclude-dir, --include, --null (-Z), --null-data (-z), --recursive (-r)
