名前

grub-mkconfig - GRUB 設定ファイルの生成  

書式

grub-mkconfig [,OPTION/]  

説明

grub 設定ファイルを生成します。
-o, --output=,FILE/
生成された設定を FILE に出力します。 [デフォルト=stdout]
-h, --help
メッセージを表示して終了します。
-V, --version
バージョン情報を表示して終了します。
 

バグ報告

バグ報告は <bug-grub@gnu.org> までお願いします。  

関連項目

grub-install(8)

grub-mkconfig の完全なドキュメントは Texinfo マニュアルとしてメンテナンスされています。info と grub-mkconfig の両プログラムが適切にインストールされていれば、以下のコマンド

info grub-mkconfig

を実行して完全なマニュアルを参照できます。
