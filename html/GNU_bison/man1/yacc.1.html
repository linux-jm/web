名前

yacc - GNU プロジェクトパーサージェネレーター  

書式

yacc [OPTION]... FILE  

説明

Yacc (Yet Another Compiler Compiler) はパーサージェネレーターです。 本バージョンは bison(1) に対する単純なラッパーです。 上位互換モードを有効にするために、 本ラッパーは -y, --yacc を受け渡します。 詳しくは bison(1) を参照してください。
