domainname

Section: Maintenance Commands (8)
Updated: May 1999

名前

domainname - 現在のドメイン名を設定・表示する
nisdomainname - 現在の NIS(YP) ドメイン名を設定・表示する
ypdomainname - 現在の NIS(YP) ドメイン名を設定・表示する  

書式

domainname [ domain ]
nisdomainname [ domain ]
ypdomainname [ domain ]

 

説明

引数なしで実行すると、 domainname は現在のシステムの NIS ドメイン名を表示する。 domainname は getdomainname(2) 関数を用いてドメイン名を取得し、 nisdomainname と ypdomainname は yp_get_default_domain(3) を用いる。

スーパーユーザーだけが、ドメイン名を設定することができる。ドメイン名は domain の引き数で指定する。ここで説明している 3 つのプログラムは いずれもドメインの名の設定に setdomainname(2) を用いる。  

関連項目

getdomainname(2), setdomainname(2), yp_get_default_domain(3), ypbind(8), ypcat(1), ypmatch(1), ypwhich(1), yppoll(8), ypserv(8), yset(8)
