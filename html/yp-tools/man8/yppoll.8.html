名前

yppoll - NIS マップのバージョンとマスターサーバーを返す  

書式

yppoll [ -h host ] [ -d domain ] mapname

 

説明

yppoll

 

オプション

-h host
host で走っている NIS サーバープロセスに問い合わせを行い、 mapname に対する情報を取得する。 host が指定されなかった場合は、 ypwhich(1) によって返されるデフォルトのサーバーに問い合わせを行う。
-d domain
domainname(1) が返すデフォルトのドメイン以外を指定する。
 

関連項目

domainname(8), ypbind(8), ypcat(8), ypmatch(1), ypserv(8), yset(8), ypwhich(1)

 

作者

yppoll は yp-tools パッケージの一部である。 yp-tools パッケージは Thorsten Kukuk <kukuk@suse.de> によって書かれた。
