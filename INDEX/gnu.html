JM インデックス (GNU)

GNU Project によって配布されているマニュアルページです。

最新版の翻訳は 太字 で、 旧版の翻訳は 細字 で示します。 リンクのないページはまだ翻訳されていません。

作業状況は 翻訳作業インデックス をご覧ください。

gnumaniak は Ragnar Hojland Espinosa 氏によって 配布 されているマニュアルパッケージです。 最近 GNU では info ファイルに力を入れており、 マニュアルページは簡略化される傾向にありますが、 gnumaniak はこれを補うべく作成されました。 JM Project では、今後 GNU のマニュアルに関しては gnumaniak のものを優先して翻訳していく方針になっています。
