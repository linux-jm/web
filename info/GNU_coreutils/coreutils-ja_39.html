6.6 sha1sum: SHA-1 ダイジェストの表示、または検査をする

sha1sum は、指定された各 file の 160-bit チェックサムを計算する。 このコマンドの使用法やオプションは、md5sum と全く同じである。 See section md5sum: MD5 ダイジェストの表示、または検査をする.

注意: SHA-1 ダイジェストは MD5 より安全であり、コリジョン (collision、衝突。別のファイルが同一の指紋を持つこと) が起きたという話を聞いたことはない。 しかしながら、大量の — と言っても非現実的なほどではない — リソースがあれば、 コリジョンを作り出せることがわかっている。この理由から、 SHA-1 は、もっと安全な SHA-2 ハッシュ・アルゴリズムに徐々に移行すべきだと、一般に考えられている。 See section sha2 utilities: SHA-2 ダイジェストの表示、または検査をする.
